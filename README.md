# Hospital Management System

A Hospital Management System built using Java, JDBC, and MySQL.

## Features

* Add Patient
* View Patients
* View Doctors
* Book Appointment
* Check Doctor Availability

## Technologies Used

* Java
* JDBC
* MySQL
* Maven
* IntelliJ IDEA

## Database Tables

* patients
* doctor
* appointment

## Project Structure

* patients.java
* doctor.java
* appointment.java
* HospitalManagementSystem.java
